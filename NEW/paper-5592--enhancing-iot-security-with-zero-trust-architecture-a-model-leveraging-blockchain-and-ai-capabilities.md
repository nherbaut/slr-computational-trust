  title: 'Enhancing IoT Security with Zero-Trust Architecture: A Model Leveraging Blockchain and AI Capabilities'
  authors: Tarun Kumar Singh, T N Nisha
  abstract: The IoT ecosystem consists of many interconnected devices, like sensors, actuators, and smart systems, integral
    parts of many industries and their everyday applications. Operating with limited computational power and memory, these
    devices are highly vulnerable to cyber-attacks. Traditional Perimeter security systems no longer suffice to defend against
    these vulnerabilities due to their implicit trust models. This study describes the implementation of a Zero Trust Architecture
    (ZTA) for the Internet of Things (IoT) as a critical solution to the rapidly increasing cybersecurity threats in IoT environments.
    This study aims to develop an architecture that provides continuous and autonomous security for IoT devices and a scalable,
    AI-driven solution for securing IoT systems. The added blockchain-enabled trust mechanisms guarantee the integrity and
    authenticity of data across the IoT network. The research deploys proof-of-concept Zero Trust architecture for IoT environments
    that can be adapted for various industries, with significant security improvements. This architecture will provide a proactive
    and scalable solution for managing IoT security by automating trust evaluation and policy enforcement processes.
