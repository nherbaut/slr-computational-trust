  title: Artificital Intelligence-Assisted Blockchain-Software Defined Cloud-Based Secure Data Exchange in Industrial IoT
    Frameworks
  authors: R. Vadivel, Shafiq Sharhrah, P. Rajeswari, John Peter K, Raunak Kumar
  abstract: The rapid evolution of Industrial Internet of Things (IIoT) systems has introduced complex challenges related
    to data security, trust management, and reliable cloud integration. However, the existing Distributed Block Software Defined
    Cloud (DistB-SDCloud) with centralized architectures and static rule-based security models are inadequate for the dynamic,
    large-scale, and heterogeneous environments of Industry 4.0. To address these limitations, this paper proposes an Artificial
    Intelligence-Assisted Blockchain-SDCloud (AI-BSDCloud) Framework that employs SDN, Federated AI, and Blockchain technologies
    to provide a secure, scalable, and intelligent IIoT-cloud infrastructure. The proposed framework is a multi-layered framework
    where SDN is used for programmable flow control and isolation of devices. Next, a BlockChain is used to guarantee decentralization
    and maintain immutable transaction history, and AI assistive trust model to dynamically learn device behaviors via federated
    learning with anomaly detection. Additionally, smart contracts are used to direct access control and check logging capabilities.
    The AI-enabled trust model includes attack detection transforming the threat detection from static to dynamic. The attack
    detection engine is based on deep-learning techniques that applies to detect and mitigate attacks while the IIoT system
    is running. The framework is validated in the IIoT environment simulation with integration of OpenFlow, Ethereum Proof-of-Authority
    (PoA) blockchain, and cloud based RESTful APIs. Experimental results show that AI-BSDCloud provided the throughput (19.5
    KB/s) based on number of nodes for securing industrial data exchange to improve operational resiliency.
