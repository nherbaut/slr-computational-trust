  title: AI-Driven Adaptive Trust Framework for Securing Distributed Cloud Networks
  authors: Anil Kumar Jonnalagadda, Sana Zia Hassan, Satya Manesh Veerapaneni, Nagarjuna Nellutla
  abstract: The proliferation of distributed cloud networks has introduced new attack surfaces and complex trust dependencies
    across heterogeneous environments. Traditional static trust models fail to adapt to the dynamic nature of modern cloudnative
    architectures, resulting in vulnerabilities against evolving cyber threats. We introduce an AI-based adaptive trust system
    capable of ongoing evaluation and dynamic updates of trust levels for nodes, services, and communication pathways in distributed
    cloud environments. Leveraging machine learning models, the framework dynamically adjusts trust metrics in real time,
    incorporating behavioral analytics, anomaly detection, and federated security intelligence. The proposed architecture
    integrates seamlessly with multi-cloud and edge deployments, supporting automated policy enforcement and decentralized
    trust computation. Evaluation results demonstrate significant improvements in detection accuracy, latency optimization,
    and resilience against advanced persistent threats (APTs), highlighting the framework’s potential for next-generation
    cloud security.
