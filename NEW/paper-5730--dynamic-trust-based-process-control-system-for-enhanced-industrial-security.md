  title: Dynamic Trust-based Process Control System for Enhanced Industrial Security
  authors: Sri Ramya, S. M. A. K. Azad
  abstract: The increasing complexity and interconnectivity of industrial process control systems have exposed them to unprecedented
    cybersecurity threats. Conventional security measures often fall short of safeguarding critical infrastructure against
    sophisticated attacks, leading to potential risks such as unauthorized access, data manipulation, and operational disruptions.
    To address these challenges, this paper presents a novel Dynamic Trust-Based Process Control System (DTBPCS) that integrates
    a trust model to assess entity reliability in real-time using reinforcement machine learning algorithms, DTB-PCS quantifies
    trust scores based on historical interactions, behavioural patterns, and credentials, enabling the system to differentiate
    between trustworthy and potentially malicious entities.
