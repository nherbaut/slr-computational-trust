  title: Machine Learning Integrated Zero Trust Automation with DevOps Principles for Continuous Security Enforcement
  authors: Ankur Mahida
  abstract: Modern distributed infrastructure, ongoing delivery pipelines and hybrid cloud environments have caused the existing
    models of security based on perimeters to be inadequate. The study presents the ML-ZTAF (Machine Learning Integrated Zero
    Trust Automation Framework) as the dynamic and flexible security enforcement framework based on the principles of Zero
    Trust and the operation of DevOps to make sure that verification is done continuously, behavior is observed, and policies
    are enforced automatically. The framework uses the machine learning-based anomaly detection, contextual trust scoring,
    and automated identity validation to remove implicit trust and actively reduce security threats throughout the software
    lifecycle. ML-ZTAF is an ongoing user, device, and system behavior analysis to detect deviation and add access controls
    without interrupting operational processes. Comparison of experiments with the existing Zero Trust models has shown that
    ML-ZTAF produces better detection accuracy, reduced false positive rates, and better automation maturity and better scalability.
    The findings reveal that ML-ZTAF helps companies to stay resilient in security in real time, as well as facilitate fast
    and scalable deployments based on DevOps.
