  title: User Behavior Trust Modeling in Cloud Security
  authors: Maryam Alruwaythi, Krishna Kambampaty, Kendall E. Nygard
  abstract: 'Evaluating user behavior in cloud computing infrastructure is important for both Cloud Users and Cloud Service
    Providers. The service providers must ensure the safety of users who access the cloud. User behavior can be modeled and
    employed to help assess trust and play a role in ensuring authenticity and safety of the user. In this paper, we propose
    a User Behavior Trust Model based on Fuzzy Logic (UBTMFL). In this model, we develop user history patterns and compare
    them current user behavior. The outcome of the comparison is sent to a trust computation center to calculate a user trust
    value. This model considers three types of trust: direct, history and comprehensive. Simulation results are included.'
