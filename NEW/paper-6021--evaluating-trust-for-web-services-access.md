  title: Evaluating trust for Web services access
  authors: E. S. Shamila, V. Ramachandran
  abstract: In open and dynamic computing environment anyone can publish their own services, for instance on the Web by using
    Web services as building blocks. The service accessing based on internet requires that agents can coordinate with each
    other in a trustable and effective manner. In this paper we review the field of trust-based Web service access using a
    belief model of evidence theory which proposes a service trust model, which involves active trust of service provider
    and combines direct trust and recommendation trust. Also we are using a belief model of evidence theory with belief, disbelief
    and uncertainty functions. In this paper we have critically reviewed the field of trust-based Web service access. We propose
    a model for formal specification of the trust behavior of a service that a software developer binds an agreement and publishes
    with the service. Our proposed model expands the range of service source via active trust of object, which makes the access
    path to resource more flexible.
