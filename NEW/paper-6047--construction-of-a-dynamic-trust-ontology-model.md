  title: Construction of a Dynamic Trust Ontology Model

  abstract: Ontology, which is a conceptional model to describe conceptions and the relationships of conceptions, originates
    from philosophy, is not only static but also dynamic. In PKI, static characteristics of trust model which based on CA

    adaptability, attack resistance, impartiality, convergence and expansibility.
