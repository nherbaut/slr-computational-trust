  title: Design and Evaluation of a Zero-Trust Cybersecurity Model on AWS for Hybrid Cloud Infrastructures
  authors: Ramesh Julakanti, Tejas Vepala, Sharath Reddy Gade, Narendra Balaji Medikonda, P. Chandrashekhar Reddy, Manish
    Kumar
  abstract: With the changing environment of cloud hybrid infrastructures, long held security models built on perimeters are
    no longer suitable to the contemporary cyber threats. Zero-Trust Architecture (ZTA) and its principle of never trust,
    always verify provides an active method of cybersecurity. This empirical research paper aims at developing and testing
    a Zero-Trust model of cybersecurity on Amazon Web Services (AWS) as it would suit a hybrid cloud environment merging an
    on-premise and a public cloud system. The model is built on AWS Identity and Access Management (IAM), AWS Verified Access,
    Amazon GuardDuty, and machine learning-based anomaly detection to maintain continuous authentication and active policy
    application. The model was evaluated experimentally through simulated workloads and intrusion conditions to compare the
    performance of models to existing access control models. Findings prove 23 per cent threat detection accuracy and a 35
    per cent decrease in unauthorized access occurrences than the regular systems with firewalls. The results confirm that
    combining the Zero-Trust principles and machine learning abilities improves resiliency, reduces insiders' threats, and
    offers security to hybrid cloud systems at scale.
