  title: 'ZAID: Zero Trust Architecture and AI-Driven IDS: A Machine Learning Framework for Secure Cloud Storage and Data
    Integrity'
  authors: Haseena Manzoor, Misbah Ashraf, Muhammad Saeed, Muhammad Ibrar, Majid Hussain, Abdul Rauf
  abstract: Cloud and Edge computing architectures revolutionize data storage and management by offering scalable and cost-effective
    services. However, the growing complexity of cyber threats and the reliance on exposed digital infrastructures pose major
    security challenges. Traditional Intrusion Detection Systems (IDSs) often struggle to detect sophisticated real-time attacks,
    while Machine Learning (ML)-based IDSs, though more effective, remain vulnerable to adversarial tactics. This study proposes
    an AI-driven intrusion detection framework that integrates the Random Forest (RF) algorithm with the Zero Trust Architecture
    (ZTA) principles to address these limitations. ZTA operates on the core principle of ’never trust, always verify’, enforcing
    continuous authentication of users, devices, and network activity. The framework persistently monitors and classifies
    network traffic to identify known and unknown attack patterns with high accuracy and a reduced number of false positives.
    By embedding intelligent detection into a Zero Trust model, the system enables adaptive, context-aware defenses tailored
    for dynamic cloud-native environments. As cyber threats continue to evolve, adopting ZTA is becoming essential rather
    than optional. This research aims to support cloud security professionals and stakeholders in implementing resilient,
    scalable, and proactive defense mechanisms for enhanced cloud security.
