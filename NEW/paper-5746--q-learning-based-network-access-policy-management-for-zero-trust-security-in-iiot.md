  title: Q-Learning Based Network Access Policy Management for Zero Trust Security in IIoT
  authors: Anamika Singh, Rajesh Kumar Dhanaraj, Anupam Kumar Sharma, Balamurugan Balusamy, Ganga Sharma, Shilpy Agrawal
  abstract: Network protection is a major concern with current technology development. Zero trust security is an emerging
    trend irrespective of the domain and application area. Zero trust security is a model that ensures authentic access to
    the network. In recent years, zero trust security has attained notable significance in research and real-world practices
    to accomplish the security desires of the technologies. To overcome the risk and consequences of cyber-attacks, the application
    of zero trust security is a novel approach. Various past pieces of research were made targeting on the application and
    advantages of zero trust security but are not aware of the major challenge in the efficient implementation of the technology.
    The time consumption of each node authentication and verification hinders the seamless working of zero trust. Integrating
    artificial intelligence with zero trust is an underlying approach to providing a solution to the time-delay issue. The
    paper focuses on implementing zero-trust security with artificial intelligence for reducing network access time. The artificial
    intelligence-enabled zero trust model reported that the network access time is reduced by 18% as compared to CRH-SRP and
    38% as compared to Cyber Security via Determinisms
